# Impact Cues

Treat the change as potentially SOX-relevant if it affects any of the following:

- revenue, billing, invoicing, tax, journal entries, close, consolidation, cash, refunds, or financial reports
- APIs, queues, file transfers, exports, imports, or batch jobs carrying financial data
- master data, pricing, product-to-ledger mappings, or reconciliation logic
- IAM, roles, permissions, break-glass access, service accounts, or segregation of duties
- audit logging, alerting, monitoring, retention, backup, restore, or production scheduling
- deployment workflows, environment protection, protected branches, or release approvals
- runbooks, evidence procedures, or control narratives used by reviewers or auditors

When uncertain, use `requires human control owner review` instead of forcing a scope decision.
